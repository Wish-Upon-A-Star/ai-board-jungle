## Issues Found And Resolved

- Issue: Users had to manually create the demo profiles and automation task before proving webhook-to-Notion behavior.
  Resolution: `npm run demo:github-notion-webhook` now creates the demo user, profiles, automation, and webhook call.
- Issue: Placeholder token files could accidentally be accepted as real values.
  Resolution: The demo script rejects empty values, `PASTE_...` values, `_HERE` placeholders, and the placeholder public webhook URL.
- Issue: The token template lacked the webhook secret needed to sign the same event GitHub would send.
  Resolution: Added `AI_BOARD_GITHUB_WEBHOOK_SECRET` to the desktop GitHub token template.
